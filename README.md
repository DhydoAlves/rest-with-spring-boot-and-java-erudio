# \# Rest with Spring Boot and Java - Erudio

# 

# Este projeto é parte da minha jornada de transição de carreira para a Engenharia de Software. Aqui, aplico conceitos avançados de \*\*Java\*\*, \*\*Spring Boot\*\* e \*\*Arquitetura REST\*\*, focando em boas práticas de desenvolvimento e organização de código.

# 

# \## 🚀 O que foi implementado até agora

# 

# \- \*\*Arquitetura RESTful:\*\* Criação de endpoints para manipulação de recursos.

# \- \*\*Service Layer:\*\* Implementação de lógica de negócio isolada do Controller (Injeção de Dependência).

# \- \*\*Mock de Dados:\*\* Criação de geradores de dados dinâmicos para testes de integração e performance.

# \- \*\*Tratamento de Exceções:\*\* Handlers customizados para erros específicos da aplicação.

# \- \*\*Padrões de Projeto (OOP):\*\* Uso intensivo de Encapsulamento, Herança e Polimorfismo.

# 

# \## 🛠️ Tecnologias Utilizadas

# 

# \- \*\*Linguagem:\*\* Java 17+

# \- \*\*Framework:\*\* Spring Boot 3.x

# \- \*\*Gerenciador de Dependências:\*\* Maven

# \- \*\*IDE:\*\* IntelliJ IDEA

# \- \*\*Controle de Versão:\*\* Git / GitHub
